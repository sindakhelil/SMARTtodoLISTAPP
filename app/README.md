# Smart ToDo List App

Une application Android pour gérer vos tâches au quotidien.

## Fonctionnalités
- Ajouter une tâche
- Cocher une tâche comme terminée
- Liste des tâches avec design moderne
- Items foncés et bouton "Ajouter" large
